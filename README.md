# rottiyzer
**rottiyzer:** Simple Script To Bruteforce the Rotation Nubmer in *ROT*

# How To Run
```
*Python3 Required*
# Clone this repository
git clone https://github.com/omakmoh/rottiyzer

# Go into the repository
cd rottiyzer

# Run the script
python3 rottiyzer.py
```
